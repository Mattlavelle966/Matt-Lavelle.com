GET TO KNOW ME

Hi! I’m Matthew Lavelle, a software developer and systems tester currently working as a co-op student with the Ministry of Public and Business Service Delivery and Procurement. I specialize in automation testing and QA for enterprise-level applications, where I’ve built Playwright-based regression suites, dynamic CRUD frameworks, and a professional Domain-Specific Language (DSL) for Playwright automation to streamline test creation for the ministry. I’m enrolled in the Computer Programming and Analysis program at St. Lawrence College and hold certifications in Python, JavaScript, and C++.

In my personal time, I love exploring new technologies and building creative projects. Recently, I’ve expanded into containerized environments by creating a multi node Docker setup featuring an NGINX proxy node, two separate site nodes, and a locally hosted LLaMA AI model node showcasing my interest in distributed systems and AI integration. Other projects include HeliControlSystem (Lua), a real-time helicopter control system; KeyCommander (C#), a Helldivers-inspired game; CC-TurtleNet (Lua), a distributed virtual system; and automation tools like C-BOT and linux tools like SwayBg-Randomizer and Scrap-YT. Whether it’s coding, experimenting with AI, or tackling complex challenges, I’m always looking for ways to push boundaries and create useful solutions.
